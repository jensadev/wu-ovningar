Övningar och exempel

Webbutveckling

Ett antal exempel med övningar för efterfrågade moment. Arbeta dig igenom dina intresseområden och gör tillhörande övningar. Alla delar är inte klara ännu, men de kommer!

Innan du kör igång rekommenderar jag att du installerar VSCode extensionen Live server(av Ritwick Dey), så att du kan arbeta smidigare utan att behöva ladda om osv. Installera, ladda om code och högerklicka sedan på html dokument och välj Open with live server.

Du kan spara ditt eget arbete i mappen eget_arbete och pusha det till git.
