Sass: Syntactically Awesome Style Sheets

Sass är ett verktyg för att förenkla och förbättra ditt arbete med css.

Du skriver sass istället för css (vilket påminner väldigt mycket om css) men det ger dig lite extra funktionalitet och förbättringar. Du kompilerar sedan detta till din css fil.

För att installera sass så behöver du npm under wsl eller powershell. Du kan installera detta enskilt eller med NODEjs. Enklast är att köra wsl(Windows Subsystem Linux) och helt enkelt skriva: sudo apt install npm. Arbetar du med powershell så glöm inte att du måste uppdatera din PATH.

När du har installerat npm så kan du installera sass, enklast gör du detta globalt. Under wsl skriv, sudo npm install -g sass

Du ska nu vara redo för att köra sass, testa genom att skriva sass i wsl. För att kompilera sass till css skriver du sass --watch input file/dir output file/dir
