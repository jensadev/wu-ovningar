För att göra detta så behöver vi någon form av navigation och länkar. Vi kommer även att använda oss av en ikon, så det är bra om du arbetat med övningarna för de momenten innan du gör detta.

Menyn kräver även lite Javascript. Du kopierar in länken i ditt html dokument och skapar javascriptet.

<script src="js/hamburgare.js"></script>

Uppgifter

  • Skapa den faktiska menyn, se exempel från Navigation.
  • Ändra stilen på ikonen.
  • Jobba med stilarna på navbaren, padding / margin.
  • Lägg till fler / färre breakpoints för Media queries i CSS.
  • Jobba med CSS för att ändra stilen på hamburgerikonen öppen/stängd, se javascriptet.
