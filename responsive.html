Responsiv design

Med responsiv design menas att vi skapar en sida som ändrar karaktär baserat på klientens skärmtyp och eller storlek. Sidan anpassas för att passa klienten så bra som möjligt.

För att testa din sida responsivt så är det enklast att arbeta med utvecklingsverktygen i webbläsaren. Tryck F12 eller CTRL+SHIFT+i för att öppna dessa. Du kan sedan klicka på ikonen (typ denna devices) i panelen som öppnats. Du kan där prova ett antal devices eller sätta läget till responsiv och ändra storleken manuellt.

Öppna panelen och ändra storleken på denna sida, vilka breakpoints kan du hitta baserat på bredden? Sättet vi gör detta på är genom något som heter media queries i CSS.

Den här sidan får ett antal media queries, dels från main.css för container klassen som påverkar sidans marginaler och bredd, men även från responsive.css där fäger och annat ändras för att illustrera sidans breakpoints(de bredder där värden ändras).

  • Testa fler och andra breakpoints, det finns massor av devices. Jag har utgått från Bootstraps värden.
  • Lägg till din navigation i dokumentet och se till att den fungerar responsivt.
