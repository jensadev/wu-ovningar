Pug

Pug är ett verktyg(likt SASS ) för att ändra på arbetsflödet med HTML. Pug låter dig skapa templater och återanvända kod för att förenkla ditt arbete.

För att köra Pug, så behöver du installera det. Detta görs med NPM och du behöver versionen med cli support.Pug på NPM.

För att installera kör du npm install pug-cli -g

Den här sidan och undersidorna är byggda med Pug, så kolla repot om du är intresserad.

Syntax

Pugs syntax skiljer sig något från att skriva HTML. Pug använder indentering för att indikera vilket innehåll som hör till taggarna och indentering för att nästa dem.
