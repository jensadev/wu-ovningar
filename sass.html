SASS

Syntactically Awesome Style Sheets

Sass är ett verktyg för att förenkla och förbättra ditt arbete med CSS.

Du skriver sass istället för CSS (vilket påminner väldigt mycket om CSS) men med extra funktionalitet och förbättringar. Du kompilerar sedan detta till CSS.

  • Guide för att installera WSL(Windows Subsystem Linux)
  • Guide för att sedan installera Node

För att installera SASS så behöver du NPM under WSL eller powershell. Du kan installera detta enskilt eller med Node. Enklast är att köra WSL och skriva: sudo apt install npm för att installera NPM-paketet. Arbetar du med powershell så glöm inte att du måste uppdatera din PATH.

När du har installerat NPM så kan du installera SASS, gör det globalt. Under WSL skriv, sudo npm install -g sass

Du ska nu vara redo för att köra SASS, testa genom att skriva sass i WSL. För att kompilera SASS till CSS skriver du sass --watch input file/dir:output file/dir

De här sidorna använder SASS för all CSS så det kan vara en bra startpunkt att undersöka. Du hittar filerna på GitHub .
