Webbutveckling 1 och 2

Exempel och övningar

Det här repositoryt samlar ett antal exempel över vanliga webbkomponenter och visar hur du kan koda dem. Arbeta dig igenom dina intresseområden och gör tillhörande övningar, eller hitta det du behöver och använd det. Repot är ett work in progress och hittar du något fel eller förbättring så gör gärna en pull request. Om du saknar något så skapa en issue på GitHub och berättar vad.

Innan du kör igång rekommenderar jag att du installerar VSCode extensionen Live server(av Ritwick Dey), så att du kan arbeta smidigare utan att behöva ladda om sidan. Installera, ladda om code och högerklicka sedan på html dokument och välj Open with live server.

Om du arbetar med övningar så kan spara ditt eget arbete i mappen eget_arbete och pusha det till git.

Validera din HTML och CSS.
