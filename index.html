Webbutveckling 1-2

Exempel och övningar

Det här repositoryt samlar ett antal exempel över vanliga webbkomponenter och visar hur du kan koda dem. Arbeta dig igenom dina intresseområden och gör tillhörande övningar, eller hitta det du behöver och använd det. Repot är ett work in progress och hittar du något fel eller förbättring så gör gärna en pull request. Om du saknar något eller hittar fel, så skapa en issue på GitHub och berätta vad.

Innan du kör igång rekommenderar jag att du installerar VSCode extensionen Live server, så att du förenklar ditt arbetsflöde. Installera, ladda om code och högerklicka sedan på html-dokument och välj Open with live server.

Alla sidorna innehåller en eller flera övningar för dig att arbeta med.
